# SmartBookFinder-AI
AI recommendation system with the Smart book summarizer and the features of Library management system

Frontend - HTML , CSS 
Database - SQLite
Backend - Python , Flask

SmartBookFinder AI have features like :- 
Library Management system (Books , Students , Fine , ADD the book, Delete the book)
AI recommendation system
Smart Book extracter

Execution Steps :-(Command Prompt)
pip install flask
cd path 
python app.py
